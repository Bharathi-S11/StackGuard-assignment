StackGuard Frontend Task
1. User Flow Implementation
Implement the following user authentication and navigation flow:
1.1. Page 1: Sign-In/Sign-Up (Public Route)
Develop a public page allowing users to toggle between sign-in and sign-up with input field validation.
1.2. Page 2: Configuration (Protected Route)
Upon successful authentication, redirect users to a protected configuration page (key between 100-1000 characters).
1.3. Page 3: Dashboard (Basic Page)
Create a simple dashboard page with a "Dashboard Page" heading. The dashboard page is a private route, not accessible before entering the configuration key.
